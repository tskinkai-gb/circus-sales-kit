## 1. プロジェクト概要

| 項目 | 内容 |
|---|---|
| 商材名 | BowNow（バウナウ） |
| 提供会社 | クラウドサーカス株式会社 |
| キット形態 | 単一HTMLまたは複数ファイル構成のSPA（ビルドツール不要） |
| 主な利用シーン | 営業担当者が商談中に画面共有、または顧客に直接渡す |
| 機密情報方針 | パートナー仕切り価格・解約阻止方針・稟議基準等の社内限定情報は一切含めない |

## 3. ファイル構成設計（Agent 1 への指示）

```
bownow-sales-kit/
├── index.html              # SPAシェル・ナビ・全ページHTMLを含む
├── css/
│   └── style.css           # デザイントークン（CSS変数）＋全スタイル
├── js/
│   ├── core.js             # SPA骨格・ページ切替・検索・ショートカット・モバイルメニュー
│   ├── product-config.js   # BowNow固有データ（料金・機能・スコア配点・ROI前提値）
│   └── ui.js               # アコーディオン・タブ・コピー・スコア計算・ROI計算
└── data/
    └── content.js          # コンテンツテキスト（トーク・Q&A・シナリオ・メール文面）
```

**更新しやすさの原則**：料金プラン改定時は `product-config.js` のみ編集すれば反映される。

## 4. フィットスコア診断 配点案（Agent 4 への指示）

5カテゴリ、各20点満点、合計100点。ティア判定は合計点で行う。

### カテゴリ定義と設問例

| カテゴリ | 配点 | 評価軸 |
|---|---|---|
| A. 課題の明確さ | 20点 | 「リード取れていない」「追跡できない」等の課題が明確か |
| B. Webサイト・トラフィック状況 | 20点 | サイトあり・一定PVあり・フォームあり、が揃っているか |
| C. 導入・運用のしやすさ | 20点 | IT担当がいる、社内決裁しやすい、専任でなくてもOK |
| D. 予算フィット | 20点 | フリー〜スタンダードの予算感に合っているか |
| E. 検討熱量・タイムライン | 20点 | 今期中に動きたい、比較検討中、決裁者が同席 |

### ティア判定

| スコア | ティア | ネクストアクション |
|---|---|---|
| 80〜100点 | ◎ 最優先提案 | フリートライアル即開始→スタンダード以上を提案 |
| 60〜79点 | ○ 提案適切 | フリープランで試用し課題を体感してもらう |
| 40〜59点 | △ 要育成 | 課題の言語化支援→再提案（1〜2ヶ月後） |
| 0〜39点 | ✕ 時期尚早 | 情報提供に留め、半年後に再接触 |

### 設問詳細（各カテゴリ4問×5点）

**A. 課題の明確さ（各5点）**
1. 自社サイトに訪問した企業・人物を把握できていない
2. リード（見込み客）の育成・フォローが手動・属人的になっている
3. メール配信が手作業で、開封・クリック状況を把握できていない
4. 営業とマーケティングの間で、どの案件が「ホット」か共有できていない

**B. Webサイト・トラフィック状況（各5点）**
1. 自社Webサイトが存在する
2. 月間PVが5,000以上ある（または増加傾向にある）
3. 問い合わせフォームや資料請求フォームが設置されている
4. コンテンツ（ブログ・事例・LP等）を定期的に更新している

**C. 導入・運用のしやすさ（各5点）**
1. マーケ専任でなくてもツール設定ができる人材がいる
2. 社内IT環境への追加ツール導入に大きな障壁がない
3. クラウドサービスの導入実績がある（他SaaS利用中）
4. 決裁者がMAツールの必要性を認識している（または説明済み）

**D. 予算フィット（各5点）**
1. 月額3〜6万円程度の予算が確保できる（またはできそう）
2. 初期費用10万円程度の社内稟議が通る見込みがある
3. 現在のリード獲得コスト・営業工数を削減したいという意識がある
4. まず無料で試してから判断したいというスタンスである

**E. 検討熱量・タイムライン（各5点）**
1. 今四半期または今期中に導入を検討している
2. 複数のMAツールを比較検討中である
3. 決裁者または導入キーマンが本日の商談に同席している
4. 具体的な課題（例：リード数が月X件）を数字で把握している

## 5. ROIシミュレーター 計算前提（Agent 4 への指示）

### 入力パラメータ
- 月間サイト訪問企業数（目安PV÷20）
- 月間新規リード獲得数（現状）
- リードフォロー担当者数
- 1人あたり月間フォロー工数（時間）
- 担当者の平均時給換算（社内コスト）
- 現在のメール配信ツール月額（0円可）

### 計算項目（比較する3パターン）
| パターン | 内容 |
|---|---|
| 現状維持（未導入） | 人件費コスト + 既存ツール費用 の合計 |
| BowNow スタンダード | 月額36,000円 + 初期費用100,000円（12ヶ月均し） + 人件費削減効果（20%減と仮定） |
| 他社MA（参考） | 月額100,000円〜（概算。HubSpot等の高機能帯） |

### 出力
- 年間コスト比較グラフ（スタンダードプランでの1年間総コスト vs 現状維持コスト）
- 損益分岐点（何ヶ月で元が取れるか）
- 推奨プラン判定（リード数・PV数から自動判定）

## 6. 各ページの必須コンテンツ

### P1: ウェルカム
- BowNow のキャッチコピー・一言説明
- 商談フロー図（初回ヒアリング→フィット診断→フリートライアル→有料アップ）
- 「このキットの使い方」ガイドボックス

### P2: ヒアリングシート
- 顧客基本情報入力エリア（会社名・業種・従業員数・担当者名）
- 課題ヒアリングチェックリスト（MAツール導入動機の確認）
- 現状確認（サイト有無・PV・フォーム有無・既存ツール）

### P3: フィットスコア診断
- 5カテゴリ×4問のチェックボックス
- リアルタイムスコア表示（100点満点）
- ティア判定表示（◎/○/△/✕）＋ネクストアクション

### P4: プラン比較・料金表
- フリー/スタンダード/ビジネス/プレミアム の縦横比較表
- 機能一覧チェックマーク表
- 他社MAツール・未導入との横断比較
- プラン選択ガイド（「あなたの状況に合うのはこちら」）

### P5: ROIシミュレーター
- スライダー＋数値入力UI
- 計算結果の3パターン比較表示
- 推奨プランの自動判定

### P6: 活用シナリオ集
- シナリオ①：マーケ専任者がいない中小企業
- シナリオ②：マーケ担当者がいるが手作業が多い
- シナリオ③：ブルーモンキー（CMS）と組み合わせる
- シナリオ④：CIRCUS Ads（広告）と組み合わせる
- 各シナリオに「こんな企業に合う」「導入後の変化」「推奨プラン」を記載

### P7: 横展開トーク
- ブルーモンキー顧客向けトーク（サイトはあるがリードが取れていない）
- CIRCUS Ads顧客向けトーク（広告で集客後のナーチャリングが弱い）
- 各トークに「商談トーク：」書き出し形式で具体的なセリフを記載

### P8: 懸念・反論処理Q&A
- 「無料プランで本当に使えるのか？」
- 「リード数が増えたら結局コストが上がるのでは？」
- 「導入・設定が難しそう」
- 「他社のMAツールと何が違うの？」
- 「今すぐ必要か分からない」
- 「既存のメール配信ツールがあるので不要では？」

### P9: フォローメールテンプレート
- 商談直後フォローメール（翌日送付用）
- フリープラン試用開始後フォローメール
- 有料プランへのアップグレード提案メール
- 各テンプレートにワンクリックコピーボタン
